## Overview
------------

Avasya is a NextJS project that utilizes various technologies such as React, TypeScript, and CSS to create a visually appealing and interactive web application.

## Project Structure
---------------------

The project is structured into the following directories:

* `src`: Contains the source code for the project, including components, pages, and styles.
* `public`: Contains static assets, such as images and fonts.
* `next.config.js`: Configuration file for NextJS.
* `tsconfig.json`: Configuration file for TypeScript.
* `package.json`: Configuration file for npm.

## Components
-------------

The project uses various components to build the UI. Some of the notable components include:

* `Navbar`: A navigation bar component that displays the logo and navigation links.
* `Hero`: A hero section component that displays a background image and text.
* `SubHome`: A sub-home section component that displays a loading animation and text.
* `Achievements`: An achievements section component that displays a list of achievements.

## Pages
------

The project has the following pages:

* `Home`: The home page that displays the hero section, sub-home section, and achievements section.
* `NotFound`: A 404 error page that displays a message indicating that the page does not exist.

## Styles
--------

The project uses CSS to style the components and pages. The styles are defined in the `globals.css` file and are imported into the components using the `import` statement.

## Configurations
----------------

The project has the following configurations:

* `next.config.js`: Configuration file for NextJS that defines the project settings, such as the page extensions and the build directory.
* `tsconfig.json`: Configuration file for TypeScript that defines the project settings, such as the target and the module system.

## Scripts
---------

The project has the following scripts:

* `dev`: Starts the development server.
* `build`: Builds the project for production.
* `start`: Starts the production server.
* `lint`: Lints the code using ESLint.

## Dependencies
------------

The project has the following dependencies:

* `next`: The NextJS framework.
* `react`: The React library.
* `typescript`: The TypeScript compiler.
* `@gsap/react`: The GSAP library for React.
* `gsap`: The GSAP library.
* `lenis`: The Lenis library.
